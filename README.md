## 📌 Descrição
Sistema de gerenciamento de tarefas desenvolvido com Electron, Node.js e MySQL. Permite criar, listar, atualizar e excluir tarefas.

## ⚙️ Funcionalidades
- Criar tarefa
- Listar tarefas
- Atualizar tarefa (PUT)
- Deletar tarefa

## 🛠 Tecnologias utilizadas
- Node.js
- Electron
- MySQL
- Express

## 📂 Divisão de tarefas
- Backend: rotas e conexão com banco de dados
- Banco: modelagem e SQL
- Frontend: interface do usuário
- Testes: Insomnia
- Git: commits, branches e organização
