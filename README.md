📌 Descrição

Sistema de gerenciamento de tarefas desenvolvido com Electron, Node.js e MySQL. Permite criar, listar, atualizar e excluir tarefas. O front-end foi desenvolvido de forma responsiva e interativa, com botões para concluir, editar (inline) e deletar tarefas.

⚙️ Funcionalidades

Criar tarefa

Listar tarefas

Atualizar tarefa (inline sem prompt)

Concluir tarefa

Deletar tarefa

🛠 Tecnologias utilizadas

Node.js

Electron

MySQL

Express

HTML, CSS, JavaScript (front-end interativo)
